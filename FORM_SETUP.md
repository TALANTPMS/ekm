# Настройка отправки форм

Обработчик находится в `php/formProcessor.php`.

## Адрес получателя

По умолчанию используется адрес из предоставленного рабочего архива:

`Savelprz2008@gmail.com`

Его можно заменить в `php/mail-config.php` или задать на сервере переменной:

`EKM_MAIL_TO`

Для нескольких адресов используйте список через запятую.

## SMTP

Если хостинг поддерживает стандартную PHP-функцию `mail()`, дополнительных
настроек не требуется. Для более надёжной отправки задайте на сервере:

- `EKM_SMTP_HOST`
- `EKM_SMTP_PORT` — обычно `465` или `587`
- `EKM_SMTP_USER`
- `EKM_SMTP_PASSWORD`
- `EKM_SMTP_ENCRYPTION` — `smtps` для порта 465 или `tls` для 587
- `EKM_MAIL_FROM` — адрес отправителя на домене сайта
- `EKM_MAIL_FROM_NAME` — например `ЕКМ`

SMTP-пароль намеренно не хранится в файлах сайта.

## Требования к хостингу

- PHP 8.0 или новее;
- расширение `openssl` для SMTP-шифрования;
- доступ к SMTP-серверу либо настроенная функция `mail()`.
